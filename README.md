# Economic Data Analysis

This project explores economic datasets through statistical analysis, data visualisation, and data-driven methodologies.

The objective is to identify trends, patterns, and relationships within economic data while developing practical skills in data analysis, quantitative reasoning, and evidence-based interpretation.

## Methods Used

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Statistical Analysis
- Data Visualisation
- Trend Analysis

## Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Skills Demonstrated

- Quantitative Analysis
- Data Interpretation
- Statistical Reasoning
- Problem Solving
- Scientific Communication
